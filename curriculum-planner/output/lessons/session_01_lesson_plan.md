# SESSION 1: “SEE IT & QUESTION IT”
============================================================

**Duration:** 465 minutes
**Materials:** Laptops, projector, student handouts, whiteboard markers

## LEARNING TARGETS

1. Identify misleading charts and explain what makes a visualization trustworthy vs. deceptive (LO #1, #5) _(Bloom's: Remember)_
2. Use 4 spreadsheet functions (SORT, FILTER, SUMIF/COUNTIF) to answer business questions (LO #2) _(Bloom's: Apply)_
3. Clean a messy dataset by removing duplicates, fixing formatting, and handling blanks (LO #2) _(Bloom's: Apply)_
4. Build 5 pivot tables answering different analytical questions (LO #4) _(Bloom's: Create)_
5. Create 3 appropriate chart types and evaluate which tells the story best (LO #1, #3) _(Bloom's: Create)_
6. Write a 3-sentence data brief in English: finding + significance + recommendation (LO #6, #7) _(Bloom's: Create)_

## ASSESSMENT AT A GLANCE

**Look for:** Delivery: Four levels: Read → Work → Analyze → Communicate. Five questions every analysis starts with: What happened? Why? What will happen? What should we do? How will we know? Practice: given a business scenario, which question are you answering? Self-assessment: where are you on the four levels right now?
**Formative:** CFU checks embedded every 10-15 minutes in lesson flow (see blocks below)
**Exit Ticket:** See closing section — one question per SWBAT

│ Whole Class | Bloom's: Understand / Apply | GRR: We Do
├──────────────────────────────────────────────────────────┤
│
│ **TEACHER DOES**
│ • Hands-on, follow-along.

│ **TEACHER TIP:** VLOOKUP will confuse 40-50% of the room. Use the dictionary analogy: "You search column A to find a word, then return what's in column B." Keep the syntax on screen at all times.

## DIFFERENTIATION NOTES

| Level | Strategy |
|-------|----------|
| **Struggling** | Provide a chart evaluation checklist (title accurate? axes labeled? scale honest?); Give a formula reference card with syntax + example for each function |
| **Advanced** | Find and critique a real misleading chart from current news; present the correction; Attempt nested functions (e.g., SUMIFS, INDEX-MATCH with multiple criteria); Mentor a struggling peer (teaching deepens understanding) |
| **Language Support** | Vocabulary wall with key data terms in English + local language; sentence frames for data briefs and presentations; bilingual glossary handout (data literacy, pivot table, correlation, dashboard, KPI); allow draft writing in L1 before translating to English |

## CLOSING & EXIT TICKET

1. Show a chart with a truncated y-axis. "Identify what makes this chart misleading and rewrite the title to make it honest."
2. "Using this mini-dataset, write a SUMIF formula that answers: 'What is the total revenue from Region A?'"
3. "List 3 data quality problems you should check for before analyzing any dataset."
4. "Given this business question: 'Which product category had the highest sales last quarter?' — describe which fields go in Rows, Columns, and Values."
5. "You need to show sales trends over 12 months. Which chart type do you choose and why? Name one chart type that would be a poor choice."
6. "Write a 3-sentence data brief about one finding from today's work: [What happened] + [Why it matters] + [What to do next]."
