# SESSION 1: “SEE IT & QUESTION IT”
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

**Duration:** 325 minutes
**Materials:** Laptops, projector, student handouts

## LEARNING TARGETS

1. Session 1 Contribution _(Bloom's: Apply)_
2. By the end of Session 1, students should be able to: _(Bloom's: Apply)_
3. Identify misleading charts and explain what makes a visualization trustworthy vs. deceptive (LO #1, #5) _(Bloom's: Remember)_
4. Use 4 spreadsheet functions (SORT, FILTER, SUMIF/COUNTIF) to answer business questions (LO #2) _(Bloom's: Apply)_
5. Clean a messy dataset by removing duplicates, fixing formatting, and handling blanks (LO #2) _(Bloom's: Apply)_
6. Build 5 pivot tables answering different analytical questions (LO #4) _(Bloom's: Create)_
7. Create 3 appropriate chart types and evaluate which tells the story best (LO #1, #3) _(Bloom's: Create)_
8. Write a 3-sentence data brief in English: finding + significance + recommendation (LO #6, #7) _(Bloom's: Apply)_

## ASSESSMENT AT A GLANCE

**Look for:** Delivery: Four levels: Read → Work → Analyze → Communicate. Five questions every analysis starts with: What happened? Why? What will happen? What should we do? How will we know? Practice: given a business scenario, which question are you answering? Self-assessment: where are you on the four levels right now?
**Formative:** CFU checks embedded in lesson flow (see blocks below)
**Exit Ticket:** See closing section
