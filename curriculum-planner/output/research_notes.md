## NOTES FOR CURRICULUM DESIGNERS

1. **Language considerations**: All tools listed are English-primary. Google tools (Gemini, NotebookLM, Teachable Machine) have the broadest language support. ChatGPT supports Armenian, Georgian, and Romanian (for Moldova). Claude supports many languages but check specific language quality for Armenian/Georgian.

2. **Internet requirements**: All tools require internet access. Ensure venue has reliable WiFi. Download any needed videos in advance as backup.

3. **Account requirements**: Most tools require Google accounts (which are universal). ChatGPT requires OpenAI account. Claude requires Anthropic account. Recommend participants create accounts BEFORE the bootcamp.

4. **Free tier limitations**: Most free tiers are sufficient for a single bootcamp session. If running multiple cohorts, consider whether usage limits reset monthly.

5. **CRAFT Framework is well-documented**: Multiple free resources exist. The framework (Context, Role, Action, Format, Target audience) is intuitive and does not require technical background - ideal for the target audience.

6. **MCP is cutting-edge**: As of March 2026, MCP is still relatively new. The Anthropic Skilljar course is the most accessible introduction. Keep this segment conceptual rather than hands-on for non-technical audiences.

7. **Moldova access issue**: Plan fallback tools for Moldova participants. ChatGPT and Google Gemini are confirmed available. Poe.com provides multi-model access with broader availability.
